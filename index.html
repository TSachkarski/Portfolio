Case Study — 01

AgentOS

AI-Augmented Contact Center Interface

Contact center agents are buried in cognitive load — managing live calls, tickets, schedules, and KPIs across 3–4 disconnected tools with no real-time visibility into their own performance. AgentOS was designed to solve that.

As sole designer and developer, I built 5 fully interactive screens — dashboard, performance, schedule, profile, and login — in a single HTML file with no frameworks. Every animation (sparklines, bar fills, pulse rings, card staggers) was implemented in pure CSS and evaluated in the live prototype, not estimated from static mockups.

Solo Designer + Dev HTML / CSS / JS WCAG 2.1 AA Desktop + Mobile

Case Study — 02

Forma

Curated Living Objects — Luxury E-Commerce Experience

Most luxury furniture brands get the editorial feel right, then destroy it the moment you click "Add to Cart." Forma was built to solve that — keeping the experience coherent from homepage to checkout, at every pixel of the purchase flow.

I designed and built 5 pages (Homepage, Collection, Product, About, Journal) with a full design system built before a single page was touched — a 6-colour token palette, dual-font type scale, and 3 distinct motion curves. The prototype is 3,326 lines of HTML/CSS/JS with zero dependencies, iterated through 4 scored audits from 8.3 to a final 9.1.

Solo Designer + Dev Figma · HTML/CSS/JS WCAG 2.1 AA 3,326 lines · 0 deps
